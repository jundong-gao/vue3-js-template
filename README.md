> 该模板已集成 `axios`， `element-plus` 第三方库， 可直接使用

# 安装
`npm install`
# 运行
`npm run serve`
# 打包
`npm run build`

# 目录结构
```
├── README.md
├── babel.config.js
├── public
│   ├── favicon.ico
│   └── index.html
├── src
│   ├── App.vue
│   ├── assets
│   ├── components 组件
│   │   └── HelloWorld.vue
│   ├── hooks hooks目录
│   ├── main.js 入口文件
│   ├── router 路由
│   │   └── index.js
│   ├── service axios服务配置
│   │   ├── config.js
│   │   └── index.js
│   ├── store vuex状态管理
│   │   └── index.js
│   ├── style 样式文件
│   │   ├── base.css
│   │   └── flex.css

│       ├── About.vue
│       └── Home.vue

```
